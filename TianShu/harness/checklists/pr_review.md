# PR 审核清单

本清单用于提交 PR 前的自检和 Review Agent 审核。

## 范围确认

- [ ] PR 只包含本次相关变更。
- [ ] 未提交本地数据文件、DuckDB、Parquet、CSV、Excel 临时文件或缓存。
- [ ] 未提交密钥、模型配置、本地 Agent 配置。

## 文档一致性

- [ ] 如果修改了规则，已确认规则不在多个目录重复维护。
- [ ] 如果修改了字段，已同步数据库设计文档和字段字典。
- [ ] 如果修改了枚举值（状态码/标志位/分类代码），已补中文含义或 `Human Review`。
- [ ] 如果修改了 Harness，已同步 `harness/README.md` 或相关检查清单。

## 检查结果

```powershell
python scripts\quality\run_all_checks.py
```

- [ ] Silver 数据字典一致性通过。
- [ ] 危险模式扫描通过。
- [ ] Schema 一致性检查通过。
- [ ] 回归测试通过。

## PR 描述

- [ ] 说明改了什么。
- [ ] 说明为什么改。
- [ ] 说明验证结果。
- [ ] 说明已知限制或后续事项。
