## 建模材料应覆盖

- ER 模型：用于描述事故、人员、车辆等实体关系。
- 维度建模：用于把 Silver 明细层组织成面向分析的事实表和维表。
- 星型模型：Gold 层首选模型，适合 Text2SQL 和 BI 问数。
- 雪花模型：仅在维度层级复杂、复用价值明确时使用。
- ODS/DWD/DWS/ADS：用于解释传统数仓分层，与本项目 Bronze/Silver/Gold 做概念映射。

## 与本项目的关系

TianShu 当前采用：

```text
Bronze 原始层
  ↓
Silver 标准层
  ↓
Gold 主题星型模型
  ↓
中文语义层 / Text2SQL Agent
```

其中：

- Bronze 接近 ODS。
- Silver 接近 DWD。
- Gold 明细事实和公共维表接近 DWS。
- 面向问数、指标和应用的语义视图接近 ADS。

## 使用原则

- 建模文档讲方法和设计理由。
- 数据库设计文档定义正式表结构。
- 字段字典定义正式字段含义。
- SQL 只能实现已审核的数据库设计。

如果建模文档和数据库设计文档冲突，以 `docs/warehouse/database_design/` 为准。
