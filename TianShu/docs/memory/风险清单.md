# 风险清单

本文件记录当前仍需持续关注的项目风险。风险不等于错误，但需要在进入下一阶段前确认是否已有规则、检查和测试覆盖。

## 高风险

| 风险编号 | 风险 | 影响 | 当前防线 |
|---|---|---|---|
| RISK-001 | AI 根据业务直觉新增 Bronze 不存在的字段 | 形成虚假 Silver 字段和错误 Gold 指标 | `check_silver_dictionary.py` |
| RISK-002 | Markdown、Excel、SQL、DuckDB schema 不一致 | Agent 和研发使用不同事实源 | 待补 `check_schema_consistency.py` |
| RISK-003 | 规则说明文档中的反例被危险模式扫描误报 | Harness 门禁不可信 | 待修 `check_dangerous_patterns.py` |
| RISK-004 | Silver 数据字典缺少字段来源列 | 字段血缘不可追溯 | 待扩展 `_gen_xlsx.py` |
| RISK-005 | Gold 模型尚无数据库设计入口 | Gold 规划可能绕过事实源 | 待建 `gold_database_design.md` |
