# REPL 基本问数示例

以下示例展示 `tianshu-ask` 交互式 REPL 的典型用法。

## 启动 REPL

```bash
$ tianshu-ask

TianShu Text2SQL Agent v1.0.0
输入中文问题开始问数，输入 /help 查看帮助，输入 /exit 退出

>
```

## 示例 1：简单时间范围查询

```
> 2026年1月曼哈顿每天有多少行程？

Agent 正在分析……

📊 回答：
2026年1月曼哈顿地区（Manhattan）平均每天有 45,230 次行程。
1月1日最高，达到 52,100 次；1月31日最低，为 38,400 次。
整体呈月初高、月末低的趋势。

📋 数据来源：gold.dws_daily_trip_summary
⏱ 执行时间：0.12s
```

## 示例 2：单表多指标查询

```
> 2026年1月曼哈顿每天行程量和事故量？

Agent 正在分析……

📊 回答：
2026年1月曼哈顿地区：
- 日均行程量：45,230 次
- 日均事故量：18 起
- 万次行程事故率：3.98 起/万次
行程量与事故量呈正相关（Pearson r=0.72）。

📋 数据来源：gold.dws_daily_trip_summary, gold.dws_daily_crash_summary
⏱ 执行时间：0.25s
```

## 示例 3：带过滤条件的查询

```
> 2026年Q1周末（周六周日）Bronx的日均行程量？

Agent 正在分析……

📊 回答：
2026年第一季度（1-3月）Bronx地区周末（周六、周日）的日均行程量为 38,500 次。
其中周六日均 39,200 次，周日日均 37,800 次。

📋 数据来源：gold.dws_daily_trip_summary
⏱ 执行时间：0.15s
```

## 示例 4：排序与 Top-N 查询

```
> 2026年1月行程量最高的5天是哪几天？

Agent 正在分析……

📊 回答：
2026年1月行程量最高的5天：
1. 1月2日（周六）— 52,300 次
2. 1月3日（周日）— 51,800 次
3. 1月1日（周五）— 51,200 次
4. 1月9日（周六）— 50,900 次
5. 1月10日（周日）— 50,600 次

周末（周五至周日）的行程量明显高于工作日。

📋 数据来源：gold.dws_daily_trip_summary
⏱ 执行时间：0.11s
```

## 退出 REPL

```
> /exit

再见！
```

或按 `Ctrl+C` 退出。
