## 已知限制

1. **MergedResult.rows 序列化**：跨表多计划合并后的 rows 可能包含 Python date 对象，report 写入时用 `default=str` 兜底
2. **DuckDB 类型转换**：result_summary 中的 column_types 强转为字符串，原始 DuckDB 类型对象不再保留
3. **单计划路径**：仅针对回答（answer）类型计算 ResultSummary/ChartSpec；clarification/refusal 正确保持为空
4. **离线模式**：真实 E2E runner 检测 Agent 离线状态并退出 1，不静默降级
