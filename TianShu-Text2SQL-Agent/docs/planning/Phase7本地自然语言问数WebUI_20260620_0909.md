# Phase 7 本地自然语言问数 Web UI 完成报告

## Summary

Phase 7 为 TianShu Text2SQL Agent 新增了本地 Web UI 界面。用户可通过浏览器输入中文问题，查看中文答案、SVG 图表、数据表格和元信息。UI 由 FastAPI 同源托管，纯原生 HTML/CSS/JS + 原生 SVG，零外部依赖。

## UI Architecture

- **frontend type**: 原生 HTML/CSS/JavaScript（纯手工，无框架）
- **serving mode**: FastAPI 同源托管 (`StaticFiles` + `FileResponse`)
- **external dependencies**: 零（无 React/Vue/Node/CDN/Google Fonts）
- **API endpoint**: `POST /v1/ask`（复用现有端点，不新增）
- **public contract**: Response Contract v1.0

## Page Layout

- **header**: TianShu 中文问数标题 + v1.0.0 版本徽章 + 本地模式标识 + API 状态指示器
- **Token**: password 输入框 + 连接/清空按钮 + "仅保存在当前页面内存"提示
- **question**: 多行输入框 + 提交/清空按钮 + Ctrl+Enter + 字符计数 + 示例快捷按钮
- **result**: answer/clarification/refusal/error 四种响应类型互斥渲染
- **chart**: 原生 SVG（line/bar）+ metric_card + table + 自动降级
- **metadata**: data.sources + warnings + execution_mode + contract_version + multi-plan 标记

## Token Security

- **storage**: JavaScript 闭包私有变量 `_token`（模块 IIFE 模式）
- **request header**: `X-TianShu-Token`（唯一传输方式）
- **clear behavior**: `clearToken()` 清空闭包变量，后续 ask 不再携带
- **leakage checks**: 不写 localStorage/sessionStorage/Cookie/IndexedDB/URL/console/DOM 文本

## Response Rendering

- **answer**: 中文答案文本 + ChartSpec 渲染 + 数据预览 + sources + warnings + meta
- **clarification**: 反问消息 + "请补充问题后重新提交" + 隐藏数据/图表区
- **refusal**: 拒绝原因 + 视觉拒绝状态 + 隐藏数据/图表区 + 无绕过按钮
- **error**: 错误码 + 安全消息 + request_id（无 traceback/SQL）
- **warnings**: 列表展示，⚠ 前缀
- **sources**: 列表展示，📊 前缀

## Chart Rendering

- **line**: 原生 SVG，X/Y 轴 + 网格 + 折线路径 + 数据点圆 + 图例色块 + aria-label
- **bar**: 原生 SVG，分类 X 轴 + 柱形 rect + 多 series 分组 + 图例
- **metric_card**: 大字号数值 + 指标名 + 来源
- **table**: `<table>` 元素，列名表头 + 数据行 + `—` 空值标记
- **fallback**: 未知 chart_type / series 缺失 / x-y 长度不一致 / 数据为空 / 数值异常 / warnings 禁止展示 → 降级 table + 原因提示

## XSS and CSP Review

- **DOM strategy**: `textContent` + `createElement`（零 `innerHTML` 插入后端数据）
- **innerHTML**: 未使用（仅用于清空容器 `innerHTML = ""`）
- **eval**: 零使用
- **CSP**: `default-src 'self'; script-src 'self'; style-src 'self'; img-src 'self' data:; connect-src 'self'; object-src 'none'; base-uri 'none'; frame-ancestors 'none'; form-action 'self'`
- **CSP 作用域**: 仅 `/` 和 `/assets/*` 路由，不影响 `/docs`、`/health/*`、`/v1/ask`
- **external resources**: 零外部 URL（HTML/CSS/JS 均不引用 http/https）

## Accessibility

- **labels**: 所有输入框有 `<label>`（`visually-hidden` 类用于视觉隐藏）
- **keyboard**: 所有按钮可键盘操作，Ctrl+Enter 快捷提交
- **focus**: 所有交互元素有 `:focus-visible` 样式
- **responsive**: 768px 以下单列布局
- **reduced motion**: `prefers-reduced-motion: reduce` 禁用动画

## Known Limitations

- Token 刷新后需重新输入（无持久化）
- 无会话历史
- 无流式响应（SSE/WebSocket）
- 图表仅使用 data_preview（前 10 行）
- 无查询取消
- 仅本机访问（127.0.0.1）
- 不适合公网部署
